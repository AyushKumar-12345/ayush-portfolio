# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AyushKumar-12345/ayush-portfolio.git
```

Go to the project directory

```bash
cd ayush-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
